Vultron Log Replay Demo - Quick Start

Note: The Log Replay demo uses file upload to load case logs. The JSONL files are located on your file system at:

Two-Actor Demo Logs

After running bash integration_tests/demo/run_multi_actor_integration_test.sh two-actor, you'll find logs at:

  • devlogs/two-actor/finder/urn_uuid_*-case-log.jsonl
  • devlogs/two-actor/vendor/urn_uuid_*-case-log.jsonl
  • devlogs/two-actor/case-actor/urn_uuid_*-case-log.jsonl

How to Use

  1. Run the container demo to generate logs:
    bash integration_tests/demo/run_multi_actor_integration_test.sh two-actor
  2. Open the UI demo and select "Log Replay" from the top bar
  3. Click "Choose Files" and select all three JSONL files from the devlogs folder
  4. The timeline will automatically visualize the case events

Features

  • Automatic participant detection (Finder, Vendor, CaseActor)
  • Timeline visualization with swimlanes
  • Event log panel showing all activities
  • State transitions (RM, VFD, EM, PXA states)
  • Support for multiple log files from different participants
Tip: You can upload logs from multiple runs or different demo scenarios by selecting multiple JSONL files at once.
