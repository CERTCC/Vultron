# Demo CLI

The unified Vultron demo CLI provides a single entry point for all demo
scripts via the `vultron-demo` command (DC-01-001 through DC-01-005).

## Usage

```shell
vultron-demo --help
vultron-demo receive-report
vultron-demo all
vultron-demo vultrabot cvd
```
