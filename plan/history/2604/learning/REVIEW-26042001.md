title: DR-01 Outbox reference-field dehydration
type: learning
date: 2026-04-20
source: REVIEW-26042001

The outbound adapter (outbox_handler.py) is solely responsible for
producing wire-clean ActivityStreams JSON. Core and BT nodes may construct
activities using full domain model instances in any field. The outbox handler
MUST dehydrate `actor`, `target`, `to`, `cc`, `origin`, `result`,
`instrument` to URI strings before wire delivery. `object_` is exempt.

**Promoted**: 2026-04-28 — captured in `specs/outbox.yaml` OX-07-001/002.
