title: BUG-26042201 Base-typed activity serialization drops inline subtype fields
type: learning
date: 2026-04-22
source: BUG-26042201
---

When object_is stored through a base reference annotation, plain model_dump()
serializes as base shape and silently omits subtype-only fields. Use
model_dump(..., serialize_as_any=True) in adapter paths. Regression coverage
must hit both persistence and outbound delivery boundaries.

**Promoted**: 2026-04-28 — captured in `notes/activitystreams-semantics.md`;
pitfall in `AGENTS.md`.
