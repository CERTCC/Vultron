## Issue #484 — Type narrowing: _resolve_current_participant_state()

`_resolve_current_participant_state()` was annotated as returning `tuple[Any, Any]`,
defeating downstream type checking. Fixed by narrowing to `tuple[RM, CS_vfd]` with
`isinstance` guards; added 6 unit tests. PR: #490.
