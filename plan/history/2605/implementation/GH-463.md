Extended two_actor_demo.py to cover the complete CVD lifecycle. Added fix
lifecycle (M4/M5), publication/embargo teardown (M6), and case closure (M7)
phases. Initial implementation (PR #474) used local-consistency workarounds in
`SvcAddParticipantStatusUseCase` because the Case Actor had a non-routable
URN-based ID. Those workarounds were removed in a follow-up (PR #488), which
introduced `ASGIEmitter` — an in-process ASGI adapter that delivers messages
to co-located actors without going through HTTP. Case Actor IDs were also
migrated to HTTP-routable URLs (`{base_url}/actors/case-actor-{slug}`).
PR #488, closes #463.
